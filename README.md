# 🌿 Bhagyoday APMC Website

## 📌 Project Overview

This project aims to design and develop a **commercial business website** for Bhagyoday APMC.
The platform focuses on showcasing the brand, products, and enabling seamless customer interaction.

---

## 🎯 Objectives

* Establish strong **digital branding**
* Display **product categories**
* Provide **company information**
* Enable **easy customer communication (WhatsApp integration)**
* Ensure a **clean, user-friendly UI**

---

## 🚀 Key Features

* 🏠 Home page with branding
* 📦 Product categories (3 sections)
* 🧾 About Us page
* 📞 Contact Us page
* 💬 Direct WhatsApp chat integration
* 🎨 Green-themed UI (as per client requirement)

---

## 🛠️ Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Spring Boot (Java)

---

## 🧩 Project Structure

* `frontend/` → UI and client-side logic
